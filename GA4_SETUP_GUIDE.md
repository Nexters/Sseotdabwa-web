# Google Analytics 4 (GA4) 측정 ID 발급 방법

## 1. Google Analytics 계정 생성

1. [Google Analytics](https://analytics.google.com/) 접속
2. "측정 시작" 버튼 클릭
3. Google 계정으로 로그인

## 2. 계정 설정

1. **계정 이름**: 원하는 이름 입력 (예: "살까말까")
2. 계정 데이터 공유 설정 선택 (기본값 유지 권장)
3. "다음" 클릭

## 3. 속성 설정

1. **속성 이름**: 웹사이트 이름 입력 (예: "살까말까 웹")
2. **보고 시간대**: "대한민국" 선택
3. **통화**: "대한민국 원(₩)" 선택
4. "다음" 클릭

## 4. 비즈니스 정보

1. 업종 카테고리 선택
2. 비즈니스 규모 선택
3. 사용 목적 선택 (여러 개 가능)
4. "만들기" 클릭

## 5. 데이터 스트림 설정

1. **플랫폼**: "웹" 선택
2. **웹사이트 URL**: 실제 도메인 입력 (예: `https://sseotdabwa.com`)
3. **스트림 이름**: 원하는 이름 입력 (예: "살까말까 웹사이트")
4. **향상된 측정**: 기본값 켜짐 유지 (페이지 조회수, 스크롤, 이탈 클릭 등 자동 추적)
5. "스트림 만들기" 클릭

## 6. 측정 ID 확인

데이터 스트림이 생성되면 **측정 ID**가 표시됩니다.

- 형식: `G-XXXXXXXXXX`
- 이 ID를 복사하여 프로젝트의 `.env` 파일에 추가하세요.

## 7. 프로젝트에 측정 ID 추가

프로젝트 루트의 `.env` 파일을 열고 다음과 같이 추가:

```env
VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```

⚠️ **주의**: 실제 측정 ID로 `G-XXXXXXXXXX` 부분을 교체하세요!

## 추가 설정 (선택사항)

### 데이터 보관 기간 설정
- 속성 설정 > 데이터 설정 > 데이터 보관
- 기본 2개월 → 14개월로 변경 권장

### Google Signals 활성화
- 속성 설정 > 데이터 수집
- Google Signals 활성화 (교차 기기 추적 가능)
